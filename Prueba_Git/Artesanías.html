Aprende peletería para crear el equipamento de tus personajes de armadura mediana como el ladrón, el guardabosques y el ingeniero.

Forja las armas pesadas de cuerpo a cuerpo como la hacha, la masa, el escudo, la espada, la daga, el mandoble, la lanza y el martillo.

Forja las armas de mediana y larga distancia, como la pistola, el rifle, el arco corto, el arco largo, el arpón, la antorcha y el cuerno de guerra.

Forja las armas de mágicas de larga distancia como el foco, el báculo, el cetro y el tridente.
